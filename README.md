## 项目介绍

世界杯.skill 是一个用于 **FIFA 世界杯比赛预测** 的 ChatGPT Skill。它的设计目标很明确：

1. **只预测世界杯**，不泛化到所有足球比赛。
2. **预测前必须联网查最新数据**，避免用旧信息猜结果。
3. **必须判断比赛是否已经结束**，已结束比赛输出真实赛果，不做“赛后预测”。
4. **主输出只给一个比分**，不把用户淹没在十几个可能比分里。
5. **支持历史数据沉淀**，把每次预测和真实赛果保存下来，用于后续校准。

它不是一个“保证命中比分”的工具，而是一个把赛前信息结构化、把模型逻辑固定化、把输出格式标准化的预测工作流。

## 效果示例

### 例 1：预测一场未开赛比赛

```text
用户 ❯ 预测世界杯：阿根廷 vs 法国，输出不要复杂，主要给一个比分。

世界杯.skill ❯
## 世界杯预测：阿根廷 vs 法国

主预测比分：阿根廷 2-1 法国

参考概率：
- 胜平负：阿根廷胜 39% / 平 28% / 法国胜 33%
- 大小球：大2.5 46% / 小2.5 54%
- 双方进球：52%
- 晋级概率：阿根廷 53% / 法国 47%
- 爆冷概率：18%

简要依据：
阿根廷防守稳定性略好，法国进攻上限更高。若双方主力阵容完整，比赛更可能是小比分胶着局。

主要风险：
首发、伤病、红牌、点球和临场战术会明显改变结果。本预测是数据分析，不是投注建议。
```

### 例 2：上传赛程截图

```text
用户 ❯ 这张截图里是世界杯赛程。识别比赛，只预测还没踢的，每场只给一个比分。

世界杯.skill ❯
识别到 3 场世界杯比赛：

1. 阿根廷 vs 法国：未开赛，进入预测。
2. 巴西 vs 德国：已结束，实际比分 1-1，不做赛前预测。
3. 西班牙 vs 英格兰：未开赛，进入预测。

主预测比分：
- 阿根廷 2-1 法国
- 西班牙 1-1 英格兰
```

### 例 3：比赛已经结束

```text
用户 ❯ 预测昨天的世界杯：巴西 vs 德国。

世界杯.skill ❯
## 比赛已结束：巴西 vs 德国

实际比分：巴西 1-1 德国

这场比赛已经结束，不再做赛前预测。
可以基于赛果、xG、阵容和关键事件做复盘，并把结果写入 match_results.csv。
```

## 功能模块

| 模块 | 作用 |
|---|---|
| 多模态输入识别 | 识别文字、截图、CSV、Excel 中的世界杯比赛信息 |
| 比赛状态判断 | 判断未开赛、进行中、已结束，避免赛后预测 |
| 最新数据检索 | 联网查询赛程、阵容、伤病、排名、xG、赔率等 |
| 历史数据读取 | 使用用户上传的历史赛果、预测记录、球队参数 |
| 球队强弱评估 | 基于 Elo、FIFA 排名、近期状态、赛事强度估算基础实力 |
| 进球分布建模 | 使用 Poisson 思路生成比分、大小球、双方进球概率 |
| 阵容状态修正 | 根据首发、伤病、停赛、体能、关键球员状态修正结果 |
| 淘汰赛逻辑 | 区分常规时间比分、晋级概率、加时和点球风险 |
| 爆冷风险判断 | 结合强弱差距、市场热度、赛制压力、阵容波动判断冷门空间 |
| 结果保存回写 | 保存预测记录和真实赛果，便于后续校准 |

## 数据能力

预测时优先考虑以下因素，能查到就纳入，查不到就标注数据缺口。

### 球队层面

- FIFA 排名
- Elo 评分
- 世界杯历史表现
- 近期战绩
- 进球数、失球数
- xG、xGA
- 射门、射正、关键机会
- 控球率、传球成功率
- 定位球进攻和防守
- 高压逼抢和反击效率
- 门将扑救表现
- 防线稳定性

### 球员层面

- 预计首发
- 伤病和停赛
- 核心球员状态
- 前锋近期进球效率
- 中场创造力
- 后卫对抗和速度劣势
- 门将状态
- 替补深度
- 老将体能风险
- 新人首发不确定性

### 比赛环境

- 小组赛 / 淘汰赛 / 半决赛 / 决赛
- 是否中立场地
- 休息天数
- 长途旅行影响
- 天气和草皮
- 裁判尺度
- 历史交锋
- 战术相克
- 心理压力
- 东道主因素

### 市场与外部信号

- 胜平负赔率
- 大小球赔率
- 盘口变化
- 市场热度
- 舆论预期
- 赛前新闻突发变化

> 市场信息只作为校准，不作为投注建议。

## 工作原理

```text
输入比赛 / 截图 / 表格
        ↓
识别世界杯比赛信息
        ↓
联网确认比赛状态
        ↓
已结束 → 输出真实赛果并可回写结果
未开赛 → 收集最新球队、球员、赛程、市场数据
        ↓
读取用户上传的历史数据
        ↓
Elo 强弱评估 + Poisson 比分分布 + xG/阵容/赛制修正
        ↓
输出一个主预测比分 + 少量参考概率
        ↓
赛后保存真实赛果，校准后续预测
```
