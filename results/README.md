# 赛后复盘

这里用于保存世界杯赛后复盘和预测命中统计。

推荐结构：

```text
results/
├── README.md
├── leaderboard.md
└── 2026-06-12.md
```

## 复盘内容

每场比赛建议记录：

- 赛前前三比分预测
- 真实比分
- 是否命中 Top 1
- 是否命中 Top 3
- 胜平负是否命中
- 大小球是否命中
- 双方进球是否命中
- 预测偏差原因

## 为什么要公开复盘

公开复盘能让项目更可信：

- 不只展示预测，也展示错误
- 可以持续改进模型
- 方便社区贡献数据源
- 让 Star 用户看到项目在持续更新
