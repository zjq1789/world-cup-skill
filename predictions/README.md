# 每日预测榜

这里用于持续发布世界杯每日预测，让项目变成一个可以被追踪、复盘和讨论的开源预测实验。

## 使用方式

每个比赛日创建一个 Markdown 文件，例如：

```text
predictions/2026-06-12.md
predictions/2026-06-13.md
```

每场比赛默认输出概率最高的三个比分：

```text
1. A 队 2-1 B 队：13.6%
2. A 队 1-1 B 队：11.9%
3. A 队 1-2 B 队：8.7%
```

## 推荐字段

| 字段 | 说明 |
|---|---|
| 比赛 | 对阵双方 |
| 状态 | 未开赛 / 进行中 / 已结束 |
| 前三比分 | 加权共识后的前三个比分 |
| 胜平负 | 主胜 / 平 / 客胜概率 |
| 大小球 | 大 2.5 / 小 2.5 概率 |
| 双方进球 | BTTS 概率 |
| 数据来源 | 使用了哪些预测机构、市场、xG/Elo、历史数据 |
| 赛后结果 | 比赛结束后填写 |
| 命中情况 | 精确比分 / 胜平负 / 大小球 / 双方进球 |

## 复盘原则

- 赛前预测不要赛后修改。
- 赛后只追加真实赛果和命中情况。
- 如果赛程、阵容或数据源有误，要在复盘中说明，而不是覆盖原预测。
