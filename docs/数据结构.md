# 数据结构

本项目支持 CSV / Excel 作为历史数据输入。字段不要求一次全部具备，但字段越完整，预测校准越可靠。

## 1. match_results.csv

用于保存真实赛果。

| 字段 | 说明 | 示例 |
|---|---|---|
| match_id | 比赛唯一编号 | wc2026-001 |
| tournament | 赛事 | FIFA World Cup 2026 |
| stage | 阶段 | Group / Round of 16 / Final |
| match_date | 比赛日期 | 2026-06-14 |
| team_a | A 队 | Argentina |
| team_b | B 队 | France |
| score_a | A 队进球 | 2 |
| score_b | B 队进球 | 1 |
| aet | 是否加时 | false |
| penalties | 是否点球 | false |
| xg_a | A 队 xG | 1.8 |
| xg_b | B 队 xG | 1.2 |
| red_cards_a | A 队红牌 | 0 |
| red_cards_b | B 队红牌 | 0 |
| notes | 备注 | key player returned |

## 2. predictions.csv

用于保存历史预测。

| 字段 | 说明 | 示例 |
|---|---|---|
| prediction_id | 预测编号 | pred-001 |
| created_at | 预测时间 | 2026-06-13T20:30:00 |
| match_id | 对应比赛编号 | wc2026-001 |
| team_a | A 队 | Argentina |
| team_b | B 队 | France |
| predicted_score_a | 预测 A 队进球 | 2 |
| predicted_score_b | 预测 B 队进球 | 1 |
| win_prob_a | A 队胜率 | 0.39 |
| draw_prob | 平局概率 | 0.28 |
| win_prob_b | B 队胜率 | 0.33 |
| over_2_5 | 大 2.5 球概率 | 0.46 |
| btts | 双方进球概率 | 0.52 |
| advance_prob_a | A 队晋级概率 | 0.53 |
| upset_prob | 爆冷概率 | 0.18 |
| data_sources | 使用的数据来源摘要 | official schedule, injury news, xG |
| final_score | 赛后填写 | 2-1 |
| exact_hit | 是否命中比分 | true |

## 3. team_features.csv

用于保存球队状态参数。

| 字段 | 说明 | 示例 |
|---|---|---|
| team | 球队 | Argentina |
| elo | Elo 评分 | 2035 |
| fifa_rank | FIFA 排名 | 1 |
| attack_rating | 进攻评分 | 1.12 |
| defense_rating | 防守评分 | 0.88 |
| recent_form | 近期状态 | 0.72 |
| avg_goals_for | 场均进球 | 1.9 |
| avg_goals_against | 场均失球 | 0.8 |
| avg_xg_for | 场均 xG | 1.7 |
| avg_xg_against | 场均 xGA | 0.9 |
| injury_impact | 伤病影响 | 0.08 |
| fatigue_index | 体能压力 | 0.15 |

## 4. 数据使用原则

- 联网最新数据优先级高于旧的本地数据。
- 用户上传数据用于补充和校准，不应覆盖已确认的官方赛果。
- 缺失字段可以为空，但输出时要说明数据缺口。
- 已结束比赛必须写入真实赛果，而不是继续预测。
