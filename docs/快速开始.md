# 快速开始

## 1. 直接输入比赛

```text
预测世界杯：阿根廷 vs 法国
```

Skill 会先联网确认比赛状态：

- 如果比赛未开始：进入赛前预测。
- 如果比赛已经结束：输出真实赛果，不再伪装成赛前预测。
- 如果比赛正在进行：说明当前更适合做实时走势分析，而不是赛前预测。

## 2. 上传截图赛程

可以上传世界杯赛程截图，然后输入：

```text
识别这张截图里的世界杯比赛，只预测还没踢的比赛。
```

Skill 会提取：

- 比赛双方
- 比赛时间
- 赛事阶段
- 是否已结束
- 是否需要预测

## 3. 上传 CSV / Excel 历史数据

支持上传历史赛果、预测记录、球队参数等数据文件。例如：

```text
结合我上传的历史赛果和最新联网数据，预测世界杯：巴西 vs 德国。
```

建议文件包括：

- `match_results.csv`：真实赛果
- `predictions.csv`：历史预测记录
- `team_features.csv`：球队状态参数

## 4. 推荐输出要求

为了保持结果清晰，默认只输出一个主预测比分：

```text
主预测比分：阿根廷 2-1 法国
```

辅助指标控制在简洁范围：

- 胜平负概率
- 大小球概率
- 双方进球概率
- 晋级概率
- 爆冷概率
- 简要依据
- 主要风险

## 5. 推荐输入格式

```text
预测世界杯：球队A vs 球队B
要求：输出一个主预测比分，附带胜平负、大小球、双方进球、晋级概率、爆冷概率。
```

或者：

```text
这张截图是世界杯赛程。请识别比赛，判断是否已结束，只预测未开赛比赛，每场只给一个主预测比分。
```
