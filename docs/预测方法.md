# 预测方法

本项目推荐使用集成预测思路：让不同模型负责不同判断，再合并成一个主预测比分。

## 1. Elo 强弱评估

Elo 用于衡量两队整体实力差距。

主要影响：

- 胜平负概率
- 爆冷概率
- 晋级概率

## 2. Poisson 比分模型

Poisson 用于根据两队预期进球生成比分分布。

流程：

1. 估计 A 队预期进球 `lambda_a`
2. 估计 B 队预期进球 `lambda_b`
3. 生成 0-0 到 5-5 的比分概率
4. 汇总胜平负、大小球、双方进球概率
5. 选择概率最高且符合综合判断的主预测比分

## 3. xG 修正

如果能获取 xG 数据，用 xG 修正进攻和防守强度。

例如：

- 进球多但 xG 低：可能存在转化率回落风险
- 进球少但 xG 高：可能存在进攻回暖空间
- 失球少但 xGA 高：防守可能存在隐患

## 4. 阵容和球员状态修正

关键球员会影响预期进球。

需要考虑：

- 核心前锋是否出场
- 主力门将是否出场
- 中场组织核心是否缺阵
- 后防主力是否停赛
- 替补深度
- 体能和赛程压力

## 5. 市场校准

赔率或市场概率只作为校准信号。

不应直接照搬赔率结论，而是用于发现：

- 市场是否明显偏向某队
- 是否存在高热球队
- 是否存在潜在爆冷空间

## 6. 输出比分选择原则

虽然内部可以计算多个比分，但默认只输出一个主预测比分。

选择时优先考虑：

1. 比分概率是否靠前
2. 是否符合胜平负方向
3. 是否符合大小球倾向
4. 是否符合双方进球概率
5. 是否符合淘汰赛场景

如用户明确要求，可以最多输出三个可能比分，但默认只输出一个。
