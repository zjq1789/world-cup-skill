# 预测方法

本项目推荐使用集成预测思路：让不同模型负责不同判断，再合并成一个主预测比分。

## 1. Elo 强弱评估

Elo 用于衡量两队整体实力差距。

主要影响：

- 胜平负概率
- 爆冷概率
- 晋级概率

## 2. Poisson 比分模型

Poisson 用于根据两队预期进球生成比分分布。

流程：

1. 估计 A 队预期进球 `lambda_a`
2. 估计 B 队预期进球 `lambda_b`
3. 生成 0-0 到 6-6 的比分概率
4. 汇总胜平负、大小球、双方进球概率
5. 不直接机械选择最高单格，而是结合总进球期望、胜负方向、双方进球和大小球选择代表性比分

## 3. 为什么不能只选最高概率比分

单个比分的概率天然会偏向低比分，尤其是：

- 1-0
- 1-1
- 0-1

这不一定代表最合理的主预测比分。比如一场比赛的总进球期望已经接近 3 球，最高单格可能仍然是 1-1，但更有代表性的比分可能是 2-1、1-2、2-2、3-0 或 3-1。

因此本 Skill 增加比分校准规则：

- 如果总进球期望高于 2.7，不应轻易输出 1-0、1-1、0-1。
- 如果大 2.5 概率高于 52%，优先允许 2-1、1-2、2-2、3-0、3-1。
- 如果双方进球概率高于 55%，避免无充分理由输出 1-0 或 0-1。
- 如果强队优势明显且进攻端状态好，应考虑 2-0、2-1、3-0、3-1，而不是默认 1-0。
- 批量预测时，如果大多数比分都是 0/1 球，必须重新校准，避免保守塌缩。

## 4. xG 修正

如果能获取 xG 数据，用 xG 修正进攻和防守强度。

例如：

- 进球多但 xG 低：可能存在转化率回落风险
- 进球少但 xG 高：可能存在进攻回暖空间
- 失球少但 xGA 高：防守可能存在隐患

## 5. 阵容和球员状态修正

关键球员会影响预期进球。

需要考虑：

- 核心前锋是否出场
- 主力门将是否出场
- 中场组织核心是否缺阵
- 后防主力是否停赛
- 替补深度
- 体能和赛程压力

## 6. 市场校准

赔率或市场概率只作为校准信号。

不应直接照搬赔率结论，而是用于发现：

- 市场是否明显偏向某队
- 是否存在高热球队
- 是否存在潜在爆冷空间
- 大小球市场是否提示比赛节奏偏开放或偏保守

## 7. 输出比分选择原则

虽然内部可以计算多个比分，但默认只输出一个主预测比分。

选择时优先考虑：

1. 是否符合胜平负方向
2. 是否符合总进球期望
3. 是否符合预期净胜球
4. 是否符合大小球倾向
5. 是否符合双方进球概率
6. 是否符合淘汰赛或小组赛场景
7. 是否避免批量预测时全部塌缩为低比分

如用户明确要求，可以最多输出三个可能比分，但默认只输出一个。
