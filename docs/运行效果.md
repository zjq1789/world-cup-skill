# 运行效果

## 示例一：明确输入一场未开赛比赛

**输入：**

```text
预测世界杯：阿根廷 vs 法国。输出不要复杂，主要给一个比分。
```

**输出示例：**

```text
## 世界杯预测：阿根廷 vs 法国

主预测比分：阿根廷 2-1 法国

参考概率：
- 胜平负：阿根廷胜 39% / 平 28% / 法国胜 33%
- 大小球：大2.5 46% / 小2.5 54%
- 双方进球：52%
- 晋级概率：阿根廷 53% / 法国 47%
- 爆冷概率：18%

简要依据：
阿根廷防守稳定性略好，法国进攻上限更高。若双方主力阵容完整，比赛大概率是小比分胶着局。

主要风险：
首发、伤病、红牌、点球和临场战术会明显改变结果。本预测是数据分析，不是投注建议。
```

## 示例二：上传赛程截图

**输入：**

```text
这张截图里是世界杯赛程。帮我识别比赛，只预测还没踢的，每场只给一个比分。
```

**输出示例：**

```text
识别到 3 场世界杯比赛：

1. 阿根廷 vs 法国：未开赛，进入预测。
2. 巴西 vs 德国：已结束，实际比分 1-1，不做赛前预测。
3. 西班牙 vs 英格兰：未开赛，进入预测。

主预测比分：
- 阿根廷 2-1 法国
- 西班牙 1-1 英格兰
```

```

```

## 示例四：结合历史数据

**输入：**

```text
我上传了 match_results.csv 和 predictions.csv。请结合历史数据和最新联网数据，预测世界杯：巴西 vs 西班牙。
```

**输出示例：**

```text
## 世界杯预测：巴西 vs 西班牙

主预测比分：巴西 1-1 西班牙

参考概率：
- 胜平负：巴西胜 32% / 平 31% / 西班牙胜 37%
- 大小球：大2.5 42% / 小2.5 58%
- 双方进球：55%
- 晋级概率：巴西 48% / 西班牙 52%
- 爆冷概率：21%

简要依据：
历史数据中两队强度接近，西班牙控球稳定性略好，巴西个人能力和反击威胁更高。综合后倾向常规时间平局。

主要风险：
首发边路球员状态和防线伤病会影响最终比分。
```
