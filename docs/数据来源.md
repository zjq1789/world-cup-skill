# 数据来源

预测前必须联网查最新数据。数据来源越新，预测越可靠。

## 优先级 1：比赛状态和官方信息

必须优先确认：

- 比赛是否未开赛
- 比赛是否已经结束
- 比赛时间和赛事阶段
- 是否为淘汰赛
- 是否可能进入加时和点球

如果比赛已经结束，输出真实比分，不再做赛前预测。

## 优先级 2：球队和阵容信息

重点查询：

- 预计首发
- 伤病
- 停赛
- 核心球员状态
- 门将状态
- 赛程密度
- 旅行和休息时间

## 优先级 3：技术数据

可用于模型校准：

- 近期进球和失球
- xG / xGA
- 射门和射正
- 控球率
- 定位球表现
- 防守压迫强度
- 历史交锋

## 优先级 4：市场信息

赔率和市场概率只能作为参考，不能作为唯一依据。

用途：

- 校准胜平负概率
- 识别市场共识
- 判断爆冷风险

## 数据缺口处理

如果查不到关键数据，应明确说明，例如：

```text
数据缺口：首发阵容尚未公布，xG 数据不完整，因此比分预测置信度为中等偏低。
```

## 注意事项

- 不使用过时数据冒充最新数据。
- 不把已结束比赛当成未赛比赛预测。
- 不提供投注建议。
